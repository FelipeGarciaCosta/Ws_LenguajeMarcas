QUE SON LAS RELACIONES DE ORDEN

Una relacion es de orden si es REFLEXIVA, ANTISIMETRICA Y TRANSITIVA a la vez

Reflexiva: una relación es reflexiva cuando todos los elementos del conjunto se relacionan con si mismos

Antisimétrica: una relación es si y sólo si para cualesquiera a , b ∈ A tales que ( a , b ) ∈ R y ( b , a ) ∈ R implica que a=b

Transitiva: una relación es transitiva si y sólo si siempre que un elemento se relaciona con otro y éste último con un tercero, entonces el primero se relaciona con el tercero

Cuando se sabe que una relación es de orden, se puede usar un diagrama simplificado que se llama Diagrama de Hasse.

Una relación de orden posee: ELEMENTOS NOTABLES de un conjunto ordenado

Estos son los maximales y los minimales del conjunto

El conjunto de cotas superiores se llama Conjunto mayorante.

El conjunto de cotas inferiores se llama Conjunto minorante.

La menor de las cotas superiores, recibe el nombre de supremo.

La mayor de las cotas inferiores, recibe el nombre de ínfimo. Si el supremo pertenece a B, se llama Máximo de B. Si el ínfimo pertenece a B, se llama Mínimo de B.
